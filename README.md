# Doodle's Outfit Planner

Doodle's Outfit Planner is a Vanilla JavaScript web app for organizing wardrobe pieces and building saved outfits.

## Features

- Add, edit, and delete clothing items
- View wardrobe items in a searchable, filterable list
- Select multiple items to create an outfit
- Save outfits in a dedicated section
- Persist wardrobe and outfit data with `localStorage`
- Responsive playful UI built with HTML, CSS, and Vanilla JavaScript

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## How to Run

1. Clone or download the repository.
2. Open `index.html` in a browser.
3. Start adding clothing items and creating outfits.

## Project Notes

This project was created for a final JavaScript assignment focused on DOM manipulation, event handling, forms, data handling, and persistent user interaction.
